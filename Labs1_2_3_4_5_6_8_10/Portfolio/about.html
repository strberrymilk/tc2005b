Acerca de este Servidor

Este es un servidor Node.js puro (sin frameworks como Express) que sirve mi portafolio personal.

Características Implementadas

Servidor HTTP Puro

Implementado con Node.js nativo, sin usar Express ni otros frameworks externos.

Múltiples Rutas

Al menos 3 rutas diferentes: /, /about, /api/mensajes y más.

Manejo de Errores 404

Respuestas HTTP 404 personalizadas cuando una ruta no existe.

Formulario POST

Formulario de contacto que envía datos por POST al servidor.

Persistencia de Datos

Los mensajes del formulario se guardan en un archivo de texto (mensajes.txt).

Archivos Estáticos

Sirve CSS, JavaScript e imágenes automáticamente.

Rutas Disponibles

Ruta Método Descripción
/ GET Página principal del portafolio
/about GET Información sobre el servidor (esta página)
/api/mensajes GET Ver todos los mensajes guardados
/submit-form POST Procesa el formulario de contacto
/*.(css|js|png|jpg) GET Archivos estáticos (CSS, JS, imágenes)
